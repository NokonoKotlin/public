こちらはcssのみで動かしています。
cssのみだと簡単ですね^_^
cssで、classの中に 『animation: Name duration timing-function delay iteration-count direction fill-mode;』　というものを書いて、
その下に 『@Keyframea Name』(Nameは上のanimationで使用したもの)と書き、進行状況(%)ごとのanimationの状態を書くだけです。
以下はcssの例です。

  2. .Animation_1{
  3.     width:7%;
  4.     height:3%;
  5.     position:relative;
  6.     background-color: rgb(159, 190, 163);
  7.     border-radius: 20%;
  8.     animation: anime_1 10s 0s infinite alternate linear;
  10. }
  12. @keyframes anime_1 {
  13.     0%{
  14.         left:0%;
  15.         top: 0px;
  16.     }
  18.     25%{
  19.         left:15%;
  20.         top:200px;
  21.     }

  24.         left:65%;
  25.         top:200px;
  26.     }
