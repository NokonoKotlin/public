# ランレングス圧縮
`ドキュメント最終更新 : 2024/05/20`

vector&lt;long long> か string を (連続数 , 要素) の vector に変換する。string は文字を long long　にキャストする。  
ex. {1,1,1,1,4,3,3,3,5,5,2,2} -> {(4,1),(1,4),(3,3),(2,5),(2,2)}
